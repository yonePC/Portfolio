# 店内商品棚卸補助ツール

## サイト概要

店内の商品の棚卸を実施するときに、読み込みした場所を投稿という形で報告することで誰がどこを読み込みしたのかを管理するアプリケーションです。

### テーマを選んだ理由

自分が前職にて棚卸がうまくいかず悔しい思いをした経験から、より効率的により精度の高い棚卸を実施できるツールがあれば良いと考えました。

### ターゲットユーザ

バーコードで管理された店内や倉庫などの商品をエリアを区切ってハンディターミナル等で読み込み棚卸を行う人。

### 主な利用シーン

棚卸における商品の読み込み時
